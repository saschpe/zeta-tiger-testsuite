==== A_28525 - PEP HTTP Proxy - Step-up-Bedingung

Die Komponente HTTP Proxy MUSS dem Client eine Step-up-Authentifizierung signalisieren, wenn der erforderliche Scope im Access-Token nicht enthalten ist oder die Audience im Access-Token nicht zur Request URL passt. Die angefragte URL MUSS existieren (unterstützte Audience).
